**Author:** Rory  
**Status:** Draft

## User Story Format

```
US-[ID]: As a [user type], I want to [action], so that [outcome].
Acceptance criteria: see docs/requirements/acceptance-criteria.md
Related requirements: FR-[ID]
```

---

## Public Portfolio Stories

**US-PF-001**  
As an employer or recruiter, I want to see a clear introduction to the developer when I land on the site, so that I immediately understand who they are and what they do.  
Related: FR-PF-001

**US-PF-002**  
As an employer, I want to read about the developer's background and what they focus on, so that I can assess whether their experience matches what I am looking for.  
Related: FR-PF-002

**US-PF-003**  
As an employer, I want to see a list of the developer's technical and professional skills, so that I can quickly assess fit for a role.  
Related: FR-PF-003

**US-PF-004**  
As an employer, I want to see real project examples with descriptions and links, so that I can evaluate the quality and scope of work the developer has shipped.  
Related: FR-PF-004

**US-PF-005**  
As an employer or collaborator, I want easy access to the developer's email, GitHub, and LinkedIn, so that I can reach out or verify their public work.  
Related: FR-PF-005

**US-PF-006**  
As a mobile user, I want the portfolio to be readable and navigable on my phone, so that I can review the developer's work without needing a desktop.  
Related: FR-PF-007

## Dashboard Stories

**US-DB-001**  
As Rory, I want a dashboard home that summarises my current state (tasks, applications, balance), so that I can quickly orient myself when I open the app.  
Related: FR-DB-002

**US-DB-002**  
As Rory, I want to navigate between all dashboard sections from a persistent menu, so that I do not have to use the browser back button to move around.  
Related: FR-DB-001

**US-DB-003**  
As Rory, I want to know that my data is not being saved in this demo version, so that I am not confused when data disappears on refresh.  
Related: FR-DB-004

## Task Tracker Stories

**US-TK-001**  
As Rory, I want to create a task with a title and optional details, so that I can capture something I need to do without leaving the app.  
Related: FR-TK-002

**US-TK-002**  
As Rory, I want to see all my current tasks in a list with their status and priority, so that I can quickly scan what needs attention.  
Related: FR-TK-001

**US-TK-003**  
As Rory, I want to update a task's status and priority as my situation changes, so that my list stays accurate.  
Related: FR-TK-003

**US-TK-004**  
As Rory, I want to delete a task when it is no longer relevant, so that my list does not accumulate noise.  
Related: FR-TK-004

## Prompt Library Stories

**US-PL-001**  
As Rory, I want to save a reusable AI prompt with a title, category, and body, so that I can find and reuse it without rewriting it.  
Related: FR-PL-002, FR-PL-006

**US-PL-002**  
As Rory, I want to copy a prompt to my clipboard with one click, so that I can paste it into a chat interface immediately.  
Related: FR-PL-005

**US-PL-003**  
As Rory, I want to filter my prompt library by category, so that I can find the right prompt without scrolling through unrelated entries.  
Related: FR-PL-007

**US-PL-004**  
As Rory, I want to edit a saved prompt when I improve it, so that my library stays up to date.  
Related: FR-PL-003

## Job Application Tracker Stories

**US-JT-001**  
As Rory, I want to log a job application with the company name, role, and status, so that I do not lose track of where I have applied.  
Related: FR-JT-002

**US-JT-002**  
As Rory, I want to update the status of an application as it progresses, so that my tracker reflects reality.  
Related: FR-JT-003, FR-JT-005

**US-JT-003**  
As Rory, I want to see a follow-up date highlighted when it has arrived or passed, so that I know when to send a follow-up without manually checking every record.  
Related: FR-JT-007

**US-JT-004**  
As Rory, I want to open the original job posting from the tracker, so that I can review the role details when preparing for an interview.  
Related: FR-JT-006

## Budget Tracker Stories

**US-BT-001**  
As Rory, I want to log an income or expense entry with an amount, category, and date, so that I have a basic record of my finances.  
Related: FR-BT-002

**US-BT-002**  
As Rory, I want to see my total income, total expenses, and net balance at a glance, so that I know where I stand without calculating it manually.  
Related: FR-BT-004

**US-BT-003**  
As Rory, I want to see spending broken down by category, so that I can identify where money is going.  
Related: FR-BT-005

**US-BT-004**  
As Rory, I want to filter my entries by income or expense, so that I can focus on one side of my finances at a time.  
Related: FR-BT-006
